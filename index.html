background-image sets the background to be an image/gradient or layers of images/gradient, the first layer being the topmost layer; By default, background images are drawn below the element's borders and above its background color; background-image: <bg-image>#; <bg-image> = url(<string>) | src(<string>) | <gradient> | none none Element must be a multi-line flex container Applies to all HTML elements; Also applies to ::first-line & ::first-letter; background-color will act as fallback; https://developer.mozilla.org/en-US/docs/Web/CSS/background-image

margin shorthand CSS property for specifying the outside space surrounding the element’s border box container; it is important to note that vertical margins collapse in size; multiple values follow a clock-wise direction, starting from the top side, as appropriate margin: <margin-side>{1,4} <margin-side> = <length> | <percentage> | auto <margin-side>{1} = <margin-top> && <margin-right> && <margin-bottom> && <margin-left> <margin-side>{2} = [<margin-top> && <margin-bottom>] [<margin-right> && <margin-left>] <margin-side>{3} = <margin-top> [<margin-right> && <margin-left>] <margin-bottom> <margin-side>{4} = <margin-top> <margin-right> <margin-bottom> <margin-left> 0 element must not be of table display type that is not table, inline-table, table-caption margin is the shorthand for the constituent properties of margin-top, margin-right, margin-bottom, margin-left; Also applies to ::first-letter; https://developer.mozilla.org/en-US/docs/Web/CSS/margin
padding shorthand property for specifying space surrounding the content box within the border box; multiple values follow a clock-wise direction, starting from the top side, as appropriate padding: <padding-side>{1,4} <padding-side> = <length> | <percentage> | auto <padding-side>{1} = <padding-top> && <padding-right> && <padding-bottom> && <padding-left> <padding-side>{2} = [<padding-top> && <padding-bottom>] [<padding-right> && <padding-left>] <padding-side>{3} = <padding-top> [<padding-right> && <padding-left>] <padding-bottom> <padding-side>{4} = <padding-top> <padding-right> <padding-bottom> <padding-left> 0 element must not be of table display type that is not table, inline-table, table-caption padding is the shorthand for the constituent properties of padding-top, padding-right, padding-bottom, padding-left; Also applies to ::first-letter; https://developer.mozilla.org/en-US/docs/Web/CSS/padding
